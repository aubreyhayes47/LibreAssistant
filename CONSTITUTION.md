# LibreAssistant Constitution (Architect's Oath)

The LibreAssistant project is founded on a commitment to openness, fairness, and human-centered design.  This constitution serves as our shared oath and binding charter.

## 1. FOSS Freedoms
We guarantee the Four Essential Freedoms to every member of the community:

1. **Freedom to Run** the program for any purpose.
2. **Freedom to Study** how the program works and adapt it to your needs.
3. **Freedom to Share** copies with anyone.
4. **Freedom to Improve** the program and release those improvements so the whole community benefits.

## 2. FAccT Ethics
We pledge to design and deploy LibreAssistant in ways that respect fairness, accountability, and transparency.  All algorithms and governance decisions are documented, reviewable, and open to community oversight.

## 3. Privacy and GDPR
We adopt data-minimization principles and provide mechanisms for consent, access, correction, and deletion.  Personal data is processed only when necessary and with explicit user approval.

## 4. Human-Centered Design
All features are developed with the well‑being of end users in mind.  We empower everyone—regardless of background—to participate in co-creating the assistant.

## 5. Duty to Uphold
Contributors and maintainers are stewards of these principles.  Any change to governance or code must honor this constitution.
