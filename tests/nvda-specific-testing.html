NVDA Setup Instructions

Before starting: Ensure NVDA is running and you're familiar with basic navigation commands.

Essential NVDA Commands for Testing

  • NVDA + Space - Toggle between browse and focus mode
  • NVDA + T - Read page title
  • NVDA + D - Read entire document
  • NVDA + F7 - List all elements (links, headings, etc.)
  • NVDA + Ctrl + D - Toggle debug logging
  • H / Shift + H - Navigate by headings
  • F / Shift + F - Navigate by form fields
  • B / Shift + B - Navigate by buttons
  • L / Shift + L - Navigate by lists

NVDA Specific Features to Test

  • Browse mode vs Focus mode transitions
  • Application mode for complex widgets
  • Virtual buffer navigation
  • Live region announcements
  • Form mode automatic switching
