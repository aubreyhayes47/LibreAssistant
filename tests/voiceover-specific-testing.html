VoiceOver Setup Instructions (macOS)

Enable VoiceOver: System Preferences → Accessibility → VoiceOver, or press Cmd + F5

Essential VoiceOver Commands

  • VO + Space - Interact with controls
  • VO + Right/Left - Navigate elements
  • VO + A - Read all content
  • VO + F5 - List form controls
  • VO + U - Open rotor menu
  • VO + H - Navigate by headings
  • VO + Command + H - Heading rotor
  • VO + Shift + Down - Interact with groups
  • VO + Shift + Up - Stop interacting
Rotor Navigation: The rotor (VO + U) is VoiceOver's primary navigation tool. Use it to quickly jump between headings, links, form controls, and landmarks.
