Screen Reader User Flow Tests

Testing Instructions: This page tests complete user workflows with screen readers (NVDA, VoiceOver, JAWS). Each test flow simulates real user journeys through the LibreAssistant interface.

Screen Reader Setup Instructions

For NVDA Users:

Ensure NVDA is running. Use NVDA + Space to switch to focus mode when interacting with controls.

Key shortcuts: Tab (navigate), Space/Enter (activate), NVDA+T (read title), NVDA+D (read document)
For VoiceOver Users (macOS):

Enable VoiceOver with Cmd+F5. Use VO+Space to interact with controls.

Key shortcuts: VO+Right/Left (navigate), VO+Space (activate), VO+A (read all), VO+F5 (list form controls)
For JAWS Users:

Ensure JAWS is running. Use application mode for web controls.

Key shortcuts: Tab (navigate), Space/Enter (activate), F (list form fields), H (headings list)
